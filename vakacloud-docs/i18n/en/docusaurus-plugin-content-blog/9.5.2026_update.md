The latest update brings highly requested features for managing images and files, along with significant improvements to the system's overall clarity.
<!-- truncate -->
## Photo Galleries and Profile Pictures

We have added support for images! It is now possible to create photo galleries for children and set profile pictures (avatars), making the system more visual and easier to navigate. Staff members and parents can now also add their own profile pictures.

## Attachments for Announcements and Events

Communication is now even easier, as you can attach files directly to calendar events and announcements.

## Smoother Bulk Data Import

We have significantly improved the bulk import tool for Excel and CSV files. The system is now smarter at reading different file formats and handles special characters much more reliably. We also introduced a new, clearer menu for uploading these files.

## Clearer Texts and Instructions

We have extensively refined the texts, instructions, and error messages throughout the service. The English user interface, in particular, has seen a major overhaul to make it more logical and consistent. We have also clarified the instructions in the file management and worker panel sections.

As always, with the introduction of new features and text updates, please let our support know if you spot any missing translations or anything unusual! Your feedback helps us perfect the service.
