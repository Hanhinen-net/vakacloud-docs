The latest VakaCloud update focuses on improving care hour management, streamlining the user experience, and strengthening the security of the service.
<!-- truncate -->
## Revamped Care Hour Management

We have introduced significant improvements to how care hours are booked and managed:

* **Booking Lock Period:** A 14-day lock period has been added to care hour reservations. This requires bookings to be made well in advance, which significantly helps in planning staff resources.
* **Emergency Additions for Staff:** To handle exceptional situations, staff members now have the ability to make "emergency additions" to care hours even during the locked period.
* **New Weekly Roster:** A new, clearer weekly roster view has been added for employees to help them quickly grasp the daily schedule.

## New Admin Tools and Better Security

* **User Daycare Assignments:** A new tool has been added to the Admin Panel, allowing daycare administrators to assign users directly to a specific daycare.
* **Security:** Several security-enhancing updates have been applied to our backend systems, protecting user data more effectively than before.
