The latest VakaCloud update introduces the highly anticipated passwordless login option and new, time-saving account management tools for daycare administrators.
<!-- truncate -->
## A New and More Secure Way to Log In

We have added support for Passkeys. This means you can now securely log in to VakaCloud without a password by using your device's built-in authentication, such as a fingerprint, facial recognition, or device PIN. This not only speeds up the login process but also significantly improves security.

## Easier Account Management for Admins

Daycare administrators now have new tools for managing user accounts directly from the admin panel. Administrators can now easily:
* Send password reset links to users
* Reset a user's multi-factor authentication (MFA) if, for example, their device is lost or changed
* Update user email addresses

All these administrative actions are securely logged in the system's audit log to maintain a high level of security. In addition, we have updated the layout of the emails sent by the system to make them clearer.

## Translations and Usability

Along with the new features, we have updated the user interface and the English and Finnish translations. If you notice anything unusual with the functionality or texts, please contact our support! Your feedback is valuable to us.
